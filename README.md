<p>RESTful API to control a few things in my home</p>
<h4>Environment Variables</h4>

* DB_PASS
* DB_USER
* DB_URL
* DB_NAME
* PEPPER
* DB_USER
* MYQ_USER
* MYQ_PASS
* COUCH_HOST
* COUCH_KEY
* DIRECTV_HOST
* PUSHOVER_TOKEN
* PUSHOVER_USER
<h4>To bundle into a binary</h4>
<code>npm run bundle</code>
<h4>To run</h4>
<code>node server.js $PORT</code>
